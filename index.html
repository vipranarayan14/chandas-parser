Usage

  • Input only one pāda of a samskrita verse.
  • The input can be in either ITRANS or Devanagari script.
  • To consider the last laghu in a pāda as guru, tick the 'Ignore Last Laghu in pāda' checkbox.

About


This webapp is based on the 'Vṛttaratnākaraṃ' of Kedāra Bhaṭṭa'.
